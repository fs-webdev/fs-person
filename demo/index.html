Name Wrapper Slot Element

You can pass a named slot as a child of <fs-person> and it will be used to wrap the persons name. This is useful if you need to make the persons name a link or a heading.

Extra Details Slot Element

You can pass in an extra details element as a child of <fs-person> and it will be displayed beneath the person's displayed information. This is helpful if you need to add an extra line of information, such as if you want to include a link to let the user "view [their] relationship" with the person.
